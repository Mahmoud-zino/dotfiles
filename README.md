# Dotfiles - Purple-themed Development Environment

My personal dotfiles for Ubuntu with i3 window manager.

## Quick Setup (Fresh Ubuntu Install)
```bash
# Clone this repo
git clone https://github.com/YOUR_USERNAME/dotfiles.git ~/dotfiles
cd ~/dotfiles

# Install everything
./install-packages.sh
./install-tools.sh
./install.sh
```

## What's Included

- i3 window manager (purple theme)
- Polybar status bar
- Wezterm terminal
- Neovim with LSP
- tmux + zsh + Starship
- Rofi launcher
- Firefox with Tridactyl
